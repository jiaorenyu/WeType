## 一、项目概览

### 1.1 核心目标
打造一款**免登录、打开即用**的公众号排版工具，专注解决"格式错乱"和"移动端预览"两大痛点。

### 1.2 技术栈
| 层面 | 技术选型 |
|------|---------|
| 前端框架 | React 18 + TypeScript |
| 编辑器核心 | Slate.js |
| 样式内联 | juice |
| 样式方案 | Tailwind CSS + @tailwindcss/typography |
| 状态管理 | Zustand |
| 部署 | Vercel |

### 1.3 里程碑
- ✅ **Week 1**：项目初始化 + 富文本编辑器核心
- ✅ **Week 2**：模板系统 + 样式内联
- ✅ **Week 3**：移动端预览 + 内容类型识别
- ✅ **Week 4**：测试优化 + 部署上线

## 八、Bug 修复记录

### 2026-04-11
- **问题**：选择示例文章时，编辑器和预览区域不显示内容
- **原因**：
  1. Slate 编辑器的 `initialValue` 只在首次创建时使用，外部值变化时不会自动更新
  2. Flexbox 布局的 `flex-1` 没有正确设置容器高度
- **修复**：
  1. 在 `AppEditor` 组件中添加 `useEffect` 监听 `initialValue` 变化并同步更新
  2. 修复布局 CSS，添加 `h-[calc(100vh-80px)]` 确保正确高度
  3. 优化预览区域样式，添加 `min-h-0` 和 `flex-shrink-0`
