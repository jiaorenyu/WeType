# WriteNow

**Write, Format, Publish.**

WriteNow 是一款极简、免费、免登录的公众号排版工具，让用户在 30 秒内完成从 Markdown 到精美排版的全流程。

## 特点

- 🚀 **纯网页版**: 无需安装，打开即用
- 🔓 **免登录**: 无需注册，保护隐私
- ⚡ **极速排版**: 30秒完成从写作到发布
- 🎨 **三套主题**: 设计师精心打磨的主题
- 💻 **代码高亮**: 支持 190+ 编程语言
- 📱 **实时预览**: 模拟手机效果
- 📋 **一键复制**: 直接粘贴到公众号编辑器

## 使用方法

1. 在左侧编辑器输入 Markdown 内容
2. 选择你喜欢的主题
3. 点击「复制到公众号」按钮
4. 粘贴到公众号编辑器即可

## 开发

```bash
npm install    # 安装依赖
npm run dev    # 启动开发服务器
npm run build  # 构建生产版本
npm run preview # 预览生产版本
```

## 技术栈

- React 18 + TypeScript
- CodeMirror 6 (编辑器)
- markdown-it (Markdown 解析)
- highlight.js (代码高亮)
- juice (样式内联)
